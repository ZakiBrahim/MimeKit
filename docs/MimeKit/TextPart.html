TextPart Class

A Textual MIME part.

Syntax

public class TextPart : MimePart

Remarks

Unless overridden, all textual parts parsed by the MimeKit.MimeParser, such as text/plain or text/html, will be represented by a MimeKit.TextPart.

For more information about text media types, see section 4.1 of http://www.ietf.org/rfc/rfc2046.txt

Text Property

Gets the decoded text content.

Syntax

public string Text { get; set; }

Value

The text.

Remarks

If the charset parameter on the MimeEntity.ContentType is set, it will be used in order to convert the raw content into unicode. If that fails or if the charset parameter is not set, iso-8859-1 will be used instead.

For more control, use TextPart.GetText(System.Text.Encoding) or TextPart.GetText(string).
