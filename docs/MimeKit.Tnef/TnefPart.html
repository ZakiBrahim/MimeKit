TnefPart Class

A MIME part containing Microsoft TNEF data.

Syntax

public class TnefPart : MimeKit.MimePart

Remarks

Represents an application/ms-tnef or application/vnd.ms-tnef part.

TNEF (Transport Neutral Encapsulation Format) attachments are most often sent by Microsoft Outlook clients.

TNEF data often contains properties that map to MimeKit.MimeMessage headers. TNEF data also often contains file attachments which will be mapped to MIME parts.
