ApplicationPgpEncrypted Class

A MIME part with a Content-Type of application/pgp-encrypted.

Syntax

public class ApplicationPgpEncrypted : MimeKit.MimePart

Remarks

An application/pgp-encrypted part will typically be the first child of a MimeKit.Cryptography.MultipartEncrypted part and contains only a Version header.
